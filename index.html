I also have some personal project ideas that I hope to work on, and look forward to the challenges that those may present. I'm very happy with my choice of becoming a Computer Scientist, and am always looking forward to my furture as one.

September 2019 - Present

Being a mentor also gets me involved in presenting workshops to students. Some of the workshop topics that mentors have presented include API's, Machine Learning, Version Control, Bash, and Success in CS.

One thing that I didn't think being a Peer Montor would give to me is the opprotunity to practice my CS skills. Helping other students who are encountering problems that I once did allows me to sharpen my skills and have the fundamentals nailed down. Becoming a Peer Mentor was one of the best decisions I made at WWU.

Projects

Microshell in C

This project was built using the C language, taking advantage of C system calls, as well as using C's pointer focused design and memory allocation for storing commands in arrays. It's main use is to read commands from the command line and appropriatly execute it. The Microshell can handle many different commands from the terminal command line that most shells can handle.

This project helped me understand the UNIX operating system, gain experience writing C code, and helped me understand programming topics like pointers, system vs library calls, and memeory. You can follow the github link to see a more in depth description of the commands possible in my Microshell.

Deadwood in Java

This project is the largest project I have ever accomplished. I worked on it with another student at WWU, and it took us about 2 months to complete. It began as a text based version of the board game Deadwood, and we integrated it to work as a GUI based version. It can be played with 2-8 players.

This project helped me understand object oriented design, as well as how to work on a software in a team. The utilization of a version control software, Github, really helped us keep things in order, and was a great learning experience and gave me some insight into how a real world software could be created. This is definitly my favorite project I've made, and it gave me a lot of ideas for future projects as well.

NorthWest Books

This website originally stemmed from a final project in my MIS 314 class, and I wanted to take the project further than what was assigned. I designed the concept for the website in Adobe Xd, which I had some experience with before hand so it was fun and challenging to proactice a new technology. I also used Xampp so that I could work on the PHP functionality locally.

Once I had some basic design ideas built, I began to build the site, first using HTML5 and CSS3. I used some CSS stylings such grid and flexbox, and once I had the HTML styled, I began to implement actual functionality using PHP and SQL.

The website uses nine different tables that are read from a database using SQL queries. Rows in these tables can be crated, removed, updated, and deleted. This was my first project uisng PHP and SQL and I had a lot of fun doing it. I understand how websites like Amazon.com work, and feel confident that I could produce a function online shopping website for a company.
