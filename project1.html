About the Project

This website originally stemmed from a final project in my CS 202 class, and I wanted to actually have the website be useful to myself. I designed the concept for the website in Adobe Xd, which I had no experience with before hand so it was fun anc challenging to learn how to use a new technology.

Once I had some basic design ideas built, I began to build the site, first using HTML5 and CSS3. I used some CSS stylings such grid and flexbox, and once I had the HTML styled, I emplemented some animations using JavaScript and jQuery. Specifically for the opening and closing menu animation, the smooth scroll, and the particles using particles.js.

One of the last things I implemented was the contact form. I had little experience with PHP and SQL, so it was difficult to implement at first. It took some trial and error, but I eventually was able to implement it correctly, and it succesfully uses mySQL to send emails to me directly from the form. I plan to continue to add to this project as I create more projects and gain more experience, and I feel extremely proud of how the project looks.
